# DatabaseManager

This repository contains a simple Python script, db.py, which will create an SQLite database named "db.sqlite" using CSV files in the world.zip zip archive and find the answer to the question In what countries is used the Spanish language? Provide their full names, sorted alphabetically.

## Project Files
db.py – the main Python script
world.zip – CSV files archive

# Instructions to clone, set up, and run the script

### Clone the repository
git clone https://github.com/martinkovav/ci2.git
cd ci2/A04

### Ensure Python 3 is installed
python3 --version

### Usage
python3 db.py city.csv country.csv countrylanguage.csv
